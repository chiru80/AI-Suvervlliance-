# AI Surveillance System using YOLO and Flask

## Overview
This project implements a real-time AI surveillance system capable of detecting objects, streaming live video, and automatically storing evidence.

## Technologies Used
- Python
- YOLOv8
- OpenCV
- Flask
- NumPy

## Features
- Real-time object detection
- Live web streaming
- Automated evidence capture
- Lightweight and scalable

## How to Run
pip install -r requirements.txt
python app.py
